# Bild-Metadaten-Analyzer

## Aufgabenstellung
- Python-Programm zu einer selbstgewählten Aufgabenstellung zu implementieren und dokumentieren.
- Das Programm soll plattformübergreifend lauffähig sein, ohne kostenpflichtige Zusatzsoftware installieren zu müssen. Es genügt ein Test unter Windows oder Linux. Bitte keine Funktionen/Zusatzsoftware verwenden, die nur unter einem bestimmten System lauffähig sind.
- Eine schriftliche Ausarbeitung von mindestens 5 Seiten (Anzahl Textseiten, ohne Inhaltsverzeichnis, Deckblatt etc.) als PDF-Dokument
- Der Python-Quellcode zum Programm, ggf. mit Kommentaren
- ggf. zusätzliche Dateien, die zum Test des Programmes nötig sind
- Alle Dateien müssen zur Belegabgabe in ein ZIP-Archiv gepackt werden
- Die schriftliche Ausarbeitung sollte folgende Inhalte/Gliederungspunkte umfassen:
--> Einleitung/Motivation, Erläuterung der selbstgewählten Aufgabenstellung
--> Installations-, Konfigurations- und Bedienungsanleitung (Sind ggf. zusätzliche Python-Bibliotheken nachzuinstallieren? Was ist zum Programmbetrieb zu beachten? Wie wird das Programm bedient?)
--> Beschreibung des Programmaufbaus (Quellcode-Dateien, ggf. Klassen, Funktionen und deren Parameter etc., Welche zusätzlichen Python-Bibliotheken werden verwendet und wozu dienen sie?). Es müssen keine größeren Source-Code-Abschnitte in die Ausarbeitung eingebettet werden, kommentieren Sie stattdessen ggf. entsprechende Abschnitte im Quellcode.
--> Kurzes Fazit, Ausblick, ggf. Verbesserungs- und Erweiterungsmöglichkeiten für das Programm
